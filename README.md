# MARR — Local Chat Frontend + Optional Proxy

ملف المشروع يشتمل على:
- index.html — الواجهة
- styles.css — الأنماط
- app.js — منطق الواجهة، محادثات، تصدير/استيراد، خيارات تخزين المفتاح
- storage.js — تشفير/فك تشفير مفتاح الـ API باستخدام Web Crypto
- server.js — (اختياري) بروكسي Node.js لحفظ المفتاح على الخادم وعدم عرضه في المتصفح
- package.json — لتشغيل الخادم

تعليمات سريعة:
1. ضع جميع الملفات في مجلدٍ واحد (جذر المشروع).
2. افتح index.html في المتصفح لتجربة الواجهة المحلية.
3. لإعداد مفتاح آمن على الخادم:
   - جهّز خادم Node.js، ثم أنشئ متغير البيئة `ADMIN_TOKEN` قبل التشغيل.
   - شغّل: `ADMIN_TOKEN=secret npm start`
   - استخدم زر "رفع المفتاح إلى الخادم" في الإعدادات مع توكن الإدارة لرفع المفتاح.
4. خيار التشفير المحلي:
   - اختر "تشغيل تشفير محلي" وأدخل كلمة مرور ثم اضغط "حفظ المفتاح مشفراً".
   - كلمة المرور لا تُخزن في التخزين الدائم؛ يُمكنك إدخالها في الجلسة عند الحاجة.
5. تصدير/استيراد: يمكنك تصدير المحادثات إلى ملف JSON أو استيراد ملف محادثة.
6. الأمان: لا تحفظ المفتاح بشكل نصي في المتصفح إن أمكن؛ استخدم بروكسي خادم.

إذا رغبت أضيف:
- دعم مستخدمين متعدّدين وتسجيل دخول،
- تخزين إلى قاعدة بيانات (SQLite/Postgres),
- رفع ملفات/صور والربط مع S3،
- تلوين أفضل للشفرة أو history stream (التدفق الجزئي للردود).

أخبرني أي التحسينات تريدي تنفيذها تلوياً، وسأزودك بالملفات المعدّلة.
